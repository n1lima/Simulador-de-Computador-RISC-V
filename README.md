### Simulador de Computador RISC-V (RV32I)

Este projeto implementa um mini-computador baseado na arquitetura RISC-V RV32I, incluindo CPU, decodificador, executor de instruções, memória, barramento e um mini-assembler para montar instruções em binário.
A estrutura é totalmente modular e segue os principais conceitos vistos em aula.
